Colour Schemes

Colour Schemes that are to be used on the site are monochromatic with a focus on blacks, greys and whites.

Colour Example Hex Value Usage
#56585C For use on the background of the header section
#BFCDE1 For use on the main navigation menu buttons
#CED3DB For use on the background of the body section

Fonts

Font is standardised throughout the webpage, with only a primary and a back up font used keep the style of the webpage minimal and consistent

Font Family Link to source Usage
Philosopher https://fonts.google.com/specimen/Philosopher?query=philosopher Used for every piece of text on the webpage, nuances are documented in the Text Styles/ Structures Section
Sans Serif Font N/A Back up font in case 'Philosopher' does not work.
